# Linux System Configuration

Monorepo containing system configuration files, automation scripts, and documentation for a Pop_OS installation.

## Structure

- **`notes/`** - System documentation with hierarchical organization
- **`i3/`** - i3 window manager configuration files
- **`scripts/`** - System automation and utility scripts
- **`services/`** - Systemd service definitions
