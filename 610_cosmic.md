# 610 - COSMIC Desktop Environment

## Configuration

**Configuration Directory**: `~/.config/cosmic/`

Settings are stored in dconf/gsettings format under various COSMIC applet configurations.

## Desktop Customization

### Wallpaper & Appearance

- **Settings** → **Appearance** → Choose wallpaper, theme, accent colors

### Keyboard Shortcuts

- **Settings** → **Keyboard** → **Shortcuts** to view/customize keybindings

### Workspaces

- Right-click desktop and select **Workspaces**
- Add new workspaces with the **+** button
- Switch using **Super + Page Up/Down** or by clicking workspace indicator
