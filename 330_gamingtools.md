# 330 - Gaming Tools

## PoE - Sidekick

**Download Source**: `https://sidekick-poe.github.io/`
**Binary Location**: `~/.local/bin/sidekick/`  

Sidekick is installed locally in the user's home directory rather than system-wide.
A shortcut script is placed in `~/.local/share/applications/sidekick.desktop`.
