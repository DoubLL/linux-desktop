# 520 - COSMIC Desktop Environment

## Overview

This Pop_OS installation uses the COSMIC desktop environment, which is System76's custom desktop built to replace GNOME Shell.

## Desktop Session Info

- **Desktop Session**: COSMIC
- **XDG Desktop**: COSMIC
- **Session Type**: X11

## Configuration Location

**Primary Config Directory**: `~/.config/cosmic/`

This directory contains 32 subdirectories with various COSMIC settings and configurations.

## Related System Configuration

### Related Config Files

- **GTK Settings**: `~/.config/gtk-3.0/`, `~/.config/gtk-4.0/`
- **DBus Settings**: `~/.config/dconf/`
- **systemd User Services**: `~/.config/systemd/`

These complement COSMIC's configuration for application theming and system integration.
